
# jerry 脚手架
Vue3版脚手架

## 安装
```bash
pnpm install jerry-vue3@latest -g
```


## 创建项目
```bash
jerry create [projectName] [templateUrl]
```
- projectName: 项目名称
- templateUrl：可选，下载远程template模版，不传走脚手架内部的template <br />
    
|   序号  |             模版           | 
| :----: |            :----:          | 
|   1    |     PC版Vue3(ElementUI)    |
|   2    |    Mobile版Vue3(VantUI)    |

## 更新脚手架
```bash
jerry update
```
